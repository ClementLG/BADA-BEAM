About Bada-Beam

Bada-Beam is an interactive web application designed to transform 2D antenna polar charts into interactive 3D radiation patterns.

How It Works

Antenna manufacturers often provide radiation patterns in 2D polar charts representing the Azimuth (Horizontal) and Elevation (Vertical) planes. Bada-Beam allows you to:

  • Load your 2D polar charts directly into the browser.
  • Calibrate the charts by setting the center, the outer ring (0 dB reference), and the main lobe direction.
  • Trace the radiation pattern manually or using our Auto-Trace color detection tool.
  • Generate a comprehensive 3D volume representing the antenna's radiation characteristics.
  • Export the resulting 3D mesh as an OBJ file or take screenshots for your reports.

Project Goals

The primary goal of this tool is to provide engineers, radio amateurs, and students with an intuitive and fast way to visualize antenna directivity in three dimensions without needing complex electromagnetic simulation software.
